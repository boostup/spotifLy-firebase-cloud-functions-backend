# About this project

`spotiflybackend` is a Firebase Cloud functions backend for the [SpotifLy react frontend demo app](https://github.com/boostup/react-spotify-clone-app).

It was created using the [Firebase CLI](https://firebase.google.com/docs/hosting/quickstart).

## To install

`npm install`

## To run

`npm run serve`

or

`firebase emulators:start` to run all emulators setup in this firebase project

## Spotify OAuth / Authorization Code Flow

http://localhost:5001/spotiflybackend/eur3/login
